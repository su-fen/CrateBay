AI Hub Scope

What must be finished before v1.0

All four surfaces need product-complete workflows, not just scaffolding.

Models

Ollama status and local model listing already exist. Pull, delete, and clearer storage management are next.

Sandboxes

Lifecycle, limits, TTL metadata, and audit trail are in place. Auto-cleanup and executor integration still need to land.

MCP

Policy and security primitives exist, but the real Manager layer still needs registry, lifecycle, logs, and config export.

Assistant

Plan and execution already work. v1.0 needs deeper AI Hub integration across models, sandboxes, and MCP workflows.
